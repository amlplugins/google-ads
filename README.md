# @amlplugins/google-ads

Native `google-ads-api` SDK re-exported under the `@amlplugins` namespace.

> Google Ads (community SDK).

## Install

```bash
npm install @amlplugins/google-ads
```

## Use

```ts
import { /* named exports from google-ads-api */ } from "@amlplugins/google-ads";
// or
import sdk from "@amlplugins/google-ads";
```

Built and published by GitHub Actions on every push to `main`.

Part of the amlplugins family — see https://github.com/amlplugins.
